Event Scheduling Web App

Personal Project

Scheduler allows groups of all sizes to schedule their events on any given available date. For ease of use, scheduler allows any user to modify the event's expected attendees in order to remove the hassle of having each independent user update their own status manually. No longer are the days of struggling to coordinate events in your friend group or consistently asking each person whether they are attending or not. You can now just check the groups status with a couple of clicks!

Project Details

  • Developed a scheduling application with a Java back-end and a JavaScript, CSS, HTML5 based front-end that allows users to schedule events on any day that is available and assign attendees to their event.
  • Implemented a persistent backend system which stores events and users into MySQL database using Hibernate mappings and JDBC connector.
  • Utilized: JavaScript, HTML5, CSS, Hibernate, Java, Git, Maven, MySQL, Bootstrap, jQuery, Ajax, JDBC

TCAT Alexa Skill

Personal Project

This simple alexa skill gives you information about the next bus heading to your destination without having to unlock your phone and launch an app. Simply ask alexa when the next bus to {destination} is from {origin} and alexa will tell you if theres anything available. If there is, alexa notifies you that the next bus departs in X minutes from {bus_stop_location}. Inspiration for this app came from the fact that there have been many days I've been too lazy to check for the next bus on my iPhone. Instead, I created a script that would automate this task. Currently, I don't have any plans on continuing development of this project.

Project Details

  • Integrated Google Maps API in order to allow users to search for all origins and destinations with accuracy.
  • Utilized: Python, Json, Google Maps API, Amazon Web Service Lambda
